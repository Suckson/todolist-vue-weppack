## Build Setup

``` bash
# Download or clone it, then install dependencies
npm install

# Development environment
npm run dev

# Production environment (need to start the node service)
npm run build
npm run start
# Default listening port 3333
```

### Technology stack

1. ES6
2. vue Vue-router vuex
3. node server koa2 
4. ssr
5. api-clould
